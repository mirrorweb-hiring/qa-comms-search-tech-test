## Getting started

Install:

```bash
npm install
```

Setup (you will see your login credentials logged in your console when this command finishes running):

```bash
npm run setup
```

Run the API and UI together:

```bash
npm run dev
```

Navigate to `http://localhost:5173` and you should see the login page!
